# **App Name**: GradeX

## Core Features:

- Student Performance Prediction: Predict student's grade category, pass/fail outcome, and risk level (low/medium/high) using machine learning models. Uses prediction confidence scores.
- At-Risk Student Identification: Identify students at risk of poor performance based on prediction results. It serves as a tool to determine the urgency of intervention needed by the student, according to the AI.
- Personalized Recommendation Generation: Generate personalized academic and behavioral recommendations based on prediction results. This acts as a tool which assists in the subject-wise and behavioral improvement.
- Admin Dashboard: Admin dashboard with access to all student data, CSV upload, system-wide analytics, risk distribution, individual student dashboards, data modification, report generation, and model performance monitoring.
- Student Dashboard: Student dashboard to view own academic data, predictions, personalized recommendations, and performance trends.
- Secure Role-Based Access: Secure authentication and authorization with separate login for Admin and Student using JWT.
- Data Ingestion via CSV: Allow admins to upload student data using CSV files.

## Style Guidelines:

- Primary color: Dark blue (#30475E) for a professional, trustworthy feel, reminiscent of academic environments.
- Background color: Light gray (#F0F4F8), providing a clean, neutral backdrop that ensures readability and focus on content.
- Accent color: Teal (#3AAFA9), for highlighting interactive elements and key data points, offering a sense of clarity and engagement.
- Headline font: 'Space Grotesk' sans-serif font, giving a techy feel; Body font: 'Inter', which combines modern looks with warmth and personality.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clear and modern icons for navigation and data representation, focusing on simplicity and intuitive understanding.
- Implement a clean, organized layout with clear sections for data visualization, prediction results, and recommendations. Ensure responsive design for various screen sizes.
